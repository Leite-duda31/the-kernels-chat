# the-kernels-chat
um projeto bobinho desenvolvido quando eu queria conversar com meu amigo, porém minha mãe havia tirado meu celular.

Usando sockets, e criptografia fernet aes-128, e tailscale, apliquei os primeiros conceitos que vi sobre redes e segurança.

(projeto iniciado em : junho de 2025, ultima aturalização em: 20/07/2026, porém eu estou atualizando ele, essa é uma versão antiga).
